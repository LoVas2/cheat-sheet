<header> 
Pour contenir une introduction d’une page, d’une section, d’un article, etc. 
<footer> 
Pour contenir une conclusion d’une section 
<nav> 
Pour contenir le ruban de navigation (fil d’Ariane, breadcrumb) qui permet de naviguer au sein du site
<article> 
Pour contenir du texte « remarquable » (pas nécessairement un article de journal) 
<section> 
Pour contenir un ensemble d’éléments de même niveau 
<aside> 
<figure>	
Pour contenir une image
Permet d’insérer une légende avec <figcaption>
<audio> / <video> 
Fournit facilement l’intégration de sons et de vidéos (avec ajout des boutons de contrôle : play, pause, etc.)
Sous-balise : <source> 
<dialog> 
Signaler des conversations 
<address> 
Pour contenir les informations sur l’auteur de la section 
<time> 
Pour contenir une date 
<meter> 


Nouveaux types d’input
time / datetime / datetime-local / date / week / month : avec datepicker
email : permet la validation du format email
url : permet la validation du format url
tel : saisie d’un numéro de téléphone (à combiner avec un pattern, par ex : [0-9]{10} )
color : colorpicker

autofocus : focus au chargement
placeholder : indication dans le champ
required : indique que le champ est obligatoire
pattern : expression régulière que doit respecter le champ
autocomplete : autorise les suggestions des anciennes saisies d’un champ du même nom
